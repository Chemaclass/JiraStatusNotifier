# Email Channel

Apart of the [mandatory parameters](README.md), you will need:

### MAILER_HOST

Value example: `smtp.gmail.com`

### MAILER_PORT

Value example: `smtp.gmail.com`

### MAILER_ENCRYPTION

Value example: `ssl`

### MAILER_USERNAME

Value example: `your@email.com`

### MAILER_PASSWORD

Value example: `1he-p@$w0rd`
